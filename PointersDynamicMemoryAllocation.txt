#Pointers และ Dynamic Memory Allocation
1. Pointers and Addresses
    pointer คือ ตัวแปรที่ใช้เก็บค่าตำแหน่งบน Address ในหน่วยความจำ
    Address ตือ ที่เก็บข้อมูลบนหน่วยความจำ
2. pointer operators
    & เรียกว่า address operator เอาไว้ระบุที่อยู่ address
    * เรียกว่า dereferencing operator เอาไว้บอกค่าที่ตำแหน่งที่ตำแหน่ง
3. การจำลองการเรียกฟังก์ชันแบบ Call by Reference
    ภาษา C การส่งค่าพารามิเตอร์ไปยังฟังก์ชันผ่านการเรียกใช้ฟังก์ชันล้วนเป็น call by value 
    สามารถใช้ pointer ช่วยจำลองการส่งผ่านค่าแบบ call by reference ได้
4. การใช้ค่าคงที่กับpointers
    - เราสามารถใช้เรากันได้เพื่อไม่ให้เปลี่ยนแปลงค่า
5. นิพจน์ใน pointer และการใช้ตัวดำเนินการคณิตศาสตร์กับ pointer
    - ตัวแปร pointer สามารถเป็น operator ให้
        นิพจน์ทางคณิตศาสตร์
        นิพจน์ในการกำหนดค่า
        นิพจน์ในการเปรียบเทียบ
    - ความหมายของpointer กับคณิตศาสตร์ การที่pointerมาบวกหรือลบไม่ใช้บวกเข้าไปอีกแต่คือการบวกเข้าไปอีกเท่า
7.การจองหน่วยความจำ
    - การจองหน่วยความจำขณะโปรแกรมกำลังทำงานอยู่ ใช้คำสั่ง malloc() และ ใช้free()สำหรับคืนพื้นที่ที่ไม่ได้ใช้
    - การใช้ malloc
        data = (int *)malloc(1000 * sizeof(int));
    - การใช้ free()
        free(data);
8.Segmentation fault(core dumped)
    - จะเกิดขึ้นเมื่อทำงานบนพื้นที่ความจำที่เราไม่ได้จองไว้หรือหน่วยความจำที่นอกเหนือจากตัวแปรที่เราประกาศไว้
      โปรแกรมจะหยุดทำงานและขึ้น ข้อความนี้ขึ้นมา
